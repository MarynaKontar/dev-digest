# @devdigest/api (server) — AGENTS.md
Fastify API: imports repos/PRs, indexes repos (repo-intel), stores agents, runs the reviewer. Hexagonal: services depend on adapter interfaces resolved through a DI container.

## Stack (beyond root)
Fastify 5 (helmet · cors · rate-limit · `fastify-sse-v2`) · Drizzle ORM + `postgres` + pgvector · `fastify-type-provider-zod` (one Zod schema = request validation AND response serialization) · octokit · simple-git · ast-grep · ripgrep · dependency-cruiser · graphology.

## Commands
- `pnpm dev` (:3001) · `pnpm build` · `pnpm typecheck`
- `pnpm db:migrate` · `pnpm db:seed` (idempotent) · `pnpm db:generate`
- Tests split BY FILENAME: `*.it.test.ts` = DB-backed (testcontainers Postgres, self-skip without Docker); everything else hermetic.
  - unit: `pnpm exec vitest run --exclude '**/*.it.test.ts'`
  - integration: `pnpm exec vitest run .it.test`

## Where things live
- `platform/container.ts` — DI composition root (start here).
- `modules/<name>/` — feature plugins (routes → service → repository); registered in `modules/index.ts`.
- `adapters/` — ports (llm · github · git · codeindex · astgrep · depgraph · tokenizer · secrets · embedder); mocks in `adapters/mocks.ts`.
- `modules/reviews/run-executor.ts` — the core review flow.
- `modules/repo-intel/` — the codebase indexer.

## Conventions (non-default)
- Add a module: create `modules/<name>/routes.ts` (default Fastify plugin) + one entry in `modules/index.ts`. Static registration, not autoload.
- Routes declare Zod `params`/`body`; invalid input → 422 before handler. Don't hand-roll `Schema.parse`.
- Cross-module shared entities (agents/reviews/runs) are repositories ON the container — never reach into another module's folder.

## Gotchas
- DB schema already contains EVERY lesson's tables; most sit empty until a lesson fills them.
- Secrets are NOT in DB/env/AppConfig — they live in `~/.devdigest/secrets.json` (mode 0600), read only via `LocalSecretsProvider`. `GITHUB_TOKEN` canonical, `GITHUB_PAT` fallback. App boots with zero keys.
- Grounding gate: a finding citing a line not in the diff is dropped; the model's self-reported score is discarded and recomputed from survivors.
- Prompt-injection defense = one trusted rule (`INJECTION_GUARD`), NOT keyword scanning.

## Do-not-touch
- `db/migrations/**` (generated by drizzle-kit) · `src/vendor/shared` is the CANONICAL contracts copy — edit deliberately.
